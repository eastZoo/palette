# PALETTE
--------
<button>,<input> 태그의 기본적인 디자인을 간단하게 만드는 스타일 가이드 생성기입니다.!!🎨
추후 스타일할 수 있는 태그를 더 늘려볼 생각입니다, 팔레트에 만들어놓은 요소 클릭 시 클립보드에 css 스타일 자동복사 

(🎇 주의 아주 섬세한 디자인은 안될 수 있음@.@ , 로컬저장안됨 페이지 새로고침시 증발( 기능추가고려) )
